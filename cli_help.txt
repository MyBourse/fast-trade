Fast Trade Help

Commands
    backtest
        Runs a backtest with the given parameters. Any strat modifications can be passed at the end of the command.
        Ex: "python -m fast_trade backtest --csv="./datafile.csv/" --strat="./strat.json"
        args
            --csv: Required, path or paths to csv_file seperated by Commands
                ex: "--csv=/path/to/candlestick_data/"
                ex, multiple: "--csv=/path/to/data/file1,/path/to/data/file2"
            --strat: Required, path to strategy file, must json format
                ex: "--strat=/path/to/strat/file.json"
